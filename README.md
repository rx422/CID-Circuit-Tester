This software was implemented within the discipline "Integrated Circuits I" by the university assistant, Bîrnaz Adrian, as a teaching software tool used in the teaching process at the Faculty of Computers, Informatics and Microelectronics by students of the Applied Electronics, Microelectronics and Nanotechnologies, Biomedical Engineering study programs.
The software tool allows checking the correctness of the received logic function simplification and the correctness of the logic circuit design.
The developed software tool allows the practical verification of theoretical calculations and aims to streamline the integrated circuit design process.
The developed software tool is very useful in the students' study process.
